# CareerPath AI

AI-Enhanced Career Guidance System for Personalized Career Pathways.

---

## Overview

CareerPath AI is a web-based application that helps school and college students make informed career decisions. It provides personalized career recommendations, skill gap analysis based on users' interests, skills, and career goals.

---

## Features

- User Login and Registration
- School Career Guidance
- College Career Guidance
- Skill Gap Analysis
- Personalized Career Suggestions

---

## Workflow

```
CareerPath AI
│
├── User Authentication
│   ├── Register
│   └── Login
│
├── School Guidance
│   ├── Enter Interests
│   ├── Select Favourite Subject
│   └── Get Stream & Career Recommendation
│
├── College Guidance
│   ├── Enter Skills
│   ├── Select Technologies
│   ├── Select Certifications
│   └── Get Domain & Role Recommendation
│
├── Skill Gap Analysis
│   ├── Select Target Role
│   ├── Compare Current Skills
│   └── View Missing Skills
```
